**Midnight Guns Community Map Depot Repo**

This repository is a central location for storing community-created map pk3 files for Midnight Guns.

Contributions are handled through pull requests. Once a submission is reviewed and merged, it will be included in the repository and synchronized to active game servers when they reboot.

By submitting content to this repository, you grant Tastyspleen Studios the right to implement, modify, and redistribute the submitted packages to both game servers and players. This ensures compatibility, stability, and long-term availability of community content.

All contributors are encouraged to follow any existing naming and packaging conventions to maintain consistency across the project.

Future updates will include documentation covering the process for building and packaging maps for submission.
